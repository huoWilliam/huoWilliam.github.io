简单介绍如何使用vim编辑器

vi与vim

Linux在命令行下的文书编辑器我们需要掌握的有vi与vim。vim可以看作是vi的升级版，可以作为“程序编辑器”使用。

为什么要学习这两个软件呢？主要有以下几点原因：

  1. 所有Unix Like系统都会内置vi文书编辑器
  2. 很多个别软件的编辑接口都会主动调用vi
  3. vim具有程序编辑的能力，方便程序设计

vi的使用

vi有三种模式，分别是：

  1. 一般指令模式

    以vi打开文件后进入的默认模式。

  2. 编辑模式

    在一般指令模式下，按下“i, I, o, O, a, A, r, R”等任何一个字母之后就会进入编辑模式。如果要回到一般指令模式，只需要按下“ESC”即可。

  3. 命令行命令模式

    在一般模式当中，输入“ : / ? ”三个中的任何一个按钮，就可以将光标移动到最下面那一列。在该模式下， 可以提供你“搜寻数据”，显示行号，大量取代字符等动作，

一些常用的动作

  1. 一般指令模式下可用的动作
    • 如果想要进行多次移动的话，例如向下移动 30 列，可以使用 “30↓” 的组合按键， 亦即加上想要进行的次数（数字）后，按下动作即可！
    • 屏幕向下移动一页：[Ctrl] + [f]或[Page Down]按键
    • 屏幕向上移动一页：[Ctrl] + [b]或[Page Up] 按键
    • 移动到这一行的最前面字符处 ：数字“0”或功能键[Home]
    • 移动到这一行的最后面字符处：$ 或功能键[End]
    • 移动到这个文件的最后一行：G
    • 移动到这个文件的第一行：gg
    • 将光标移动到第n行：nG，n为数字
    • 光标向下移动n行：n\<Enter>，n为数字
    • 向光标之下寻找一个名称为 word 的字串：/word
    • 在第 n1 与 n2 行之间寻找 word1 这个字串，并将该字串取代为 word2::n1,n2s/word1/word2/g
    • 在一行中，向后删除一个字符：x
    • 在一行中，向前删除一个字符：X
    • 删除光标所在的那一整列：dd
    • 删除光标所在的向下 n 列：ndd，n为数字
    • 复制光标所在的那一行：yy
    • 将已复制的数据在光标下一行贴上：p
    • 将已复制的数据在光标上一行贴上：P
    • 还原：u
    • 重做上一个动作：[Ctrl]+r
    • 重复前一个动作：.
  2. 一般指令模式切换到编辑模式的按钮
    • 从目前光标所在处插入：i
    • 从目前光标所在的下一个字符处开始插入：a
    • 在目前光标所在的下一行处插入新的一行：o
    • 取代光标所在的那一个字符一次：r
    • 持续取代光标所在的文字，直到按下 ESC 为止：R
  3. 命令行模式下可用的按钮
    • 将编辑的数据写入硬盘文件中：:w
    • 离开 vi：:q
    • 保存并退出：:wq
    • 暂时离开 vi 到命令行界面下执行 command 的显示结果：:!command

vim的恢复功能与打开时的警告信息

当你的系统因为某些原因而导致类似死机的情况时， 还可以通过恢复功能来将你之前未及时储存的数据“救”回来。

原理是：vim在编辑文件时，会在该文件目录下创建一个.filename.swap的临时文件，用来记录你对文件所作的动作。如果你的系统因为某些原因死机了，导致你的数据尚未保存，此时就可以使用该临时文件来恢复。

当vim因为不知名原因中断当前文件的编辑后，再重新打开该文件时，会出现警告信息。解决的方式可以是，删除该临时文件，或者选择按照该临时文件进行恢复。

vim的其他功能

区块选择（visual block）

区块选择所使用的按键主要有以下几个：

  • v：字符选择，会将光标经过的地方反白选择！
  • V：行选择，会将光标经过的行反白选择！
  • [ctrl]+v：区块选择，可以用长方形的方式选择数据

之后可以选择使用按键d，按键p或按键y来对选择的区块进行删除，粘贴后复制的动作。

多文件编辑

我们可以在使用 vim指令时， 后面同时接好几个文件来同时打开多个文件。

与多文件编辑相关的按键有：

  • :n：编辑下一个文件
  • :N：编辑上一个文件
  • :files：列出目前这个vim所打开的所有文件

多窗口功能

类似于Win下的分屏功能，可以在同一个界面显示两个文件。

相关的按键如下所示：

  • :sp [filename]：打开一个新窗口，如果有加 filename， 表示在新窗口打开一个新文件，否则表示两个窗口为同一个文件内容
  • [ctrl]+w+↓：将光标移动到下方的窗口，按法是先按住[ctrl]+w，之后松开这两个按键，在按下方向键
  • [ctrl]+w+↑：将光标移动到上方的窗口
  • :q：结束当前窗口

vim的挑字补全功能

有如下几种按键组合，可以为vim中的文字进行补全：

  • [ctrl]+x -> [ctrl]+n：通过目前正在编辑的这个“文件的内容文字”作为关键字，予以补齐
  • [ctrl]+x -> [ctrl]+o：以扩展名作为语法补充，以 vim 内置的关键字，予以补齐

vim配置与记录

vim 会主动的将你曾经做过的行为登录下来，好让你下次可以轻松的作业。而这个记录动作的文件就是：~/.viminfo

对于vim中是否显示行号，是否要高亮显示搜索字符等，可以自己对vim进行配置。常见的vim配置参数有如下这些：

  • :set nu或:set nonu：即设置是否显示行号
  • :set autoindent或:set noautoindent：设置是否自动缩进排版
  • :set all：显示目前所有的环境参数设置值
  • :syntax on或:syntax off：设置是否开启语法高亮
  • :set bg=dark或:set bg=light：设置不同的背景，默认是light

但是，通过在命令行模式对vim进行配置不是长久之计，无法长久有效。如果想要一次性永久配置vim必须借助于vim的配置文件。整体vim的配置文件是：/etc/vimrc。不建议修改该文件。应该对用户的vim配置文件~/.vimrc（默认不存在，可以手动创建）进行修改。
